# 설치형 프로그램(GUI) 배포 안내

명령어·파이썬 없이 회계사가 쓸 수 있는 **데스크톱 GUI + 단독 실행파일(.exe)** 입니다.
(GUI: `afc/gui.py`, tkinter / 패키징: PyInstaller)

---

## A. 끝난 사용자(회계사)에게 — 쓰는 법

전달받은 **`금융기관조회서변환기.exe`** 하나만 있으면 됩니다. 설치 불필요.

1. `금융기관조회서변환기.exe` **더블클릭** → 창이 열립니다.
2. **① 조회서 ZIP 파일** [선택…] — 은행에서 받은 `금융기관조회서(전자)_[회사]_[날짜].zip` 선택.
3. (선택) **② 회사 예금명세** — 회사가 준 예금명세 엑셀이 있으면 선택(대사용). 없으면 비워둠.
4. **③ 결과 저장 폴더** — 기본값은 바탕화면 `금융조회_결과`. 바꿔도 됨.
5. **[변환 시작 ▶]** 클릭 → 끝나면 "결과 폴더를 열까요?" → **예**.
6. 폴더에 엑셀 6종(요약·BANK·INSURANCE·GUARANTEE·INVESTMENT·PBC대사)이 생성됩니다.

> ⚠️ **첫 실행 시 Windows 경고**: 서명 안 된 exe라 "Windows의 PC 보호" 파란 창이 뜰 수 있음 →
> **추가 정보 → 실행** 누르면 됩니다. (백신이 PyInstaller exe를 오탐할 수 있음 — 정상)

## B. 개발자(나)에게 — 빌드/재배포

### 빌드 (한 번 클릭)
프로젝트 폴더에서 **`build_exe.bat` 더블클릭** (또는 명령행):
```powershell
cd C:\Users\CJI\dev\AutomaticBankaccountConfirmation
pip install -e . pyinstaller
pyinstaller --noconfirm --onefile --windowed --name 금융기관조회서변환기 ^
  --add-data "configs;configs" --collect-all pymupdf afc\gui.py
```
→ **`dist\금융기관조회서변환기.exe`** (약 67MB) 생성. 이 파일만 복사해 배포.

### 핵심 포인트
- `--add-data "configs;configs"` : institutions/account_mapping/sections.yaml 을 exe 안에 포함.
  런타임 경로는 `afc/paths.py`가 `sys._MEIPASS`(번들) ↔ 개발폴더를 자동 분기.
- `--collect-all pymupdf` : 표 인식(find_tables) 바이너리 포함.
- `--windowed` : 콘솔창 없이 GUI만.
- 코드 변경 후 재빌드하면 됨(설정 yaml은 자동 포함).

### GUI 없이 테스트(명령행 모드)
exe(또는 `python -m afc.gui`)에 zip 경로를 인자로 주면 창 없이 변환:
```powershell
금융기관조회서변환기.exe "조회서.zip" "회사명세.xlsx(선택)"
```

## C. 더 다듬을 점 (선택)
- **아이콘** : `--icon app.ico` 로 exe 아이콘 지정.
- **설치 마법사** : Inno Setup으로 "다음→다음→설치" 인스톨러 제작(바탕화면 바로가기 자동).
- **코드 서명** : 인증서로 서명하면 Windows 경고가 사라짐(유료 인증서 필요).
- **자동 업데이트** : 추후 기능 추가 시 새 exe로 교체 배포.
